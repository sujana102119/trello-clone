## Requirements:
    Nodejs- version 16.4.0 or greater
    npm - version 8.3.1 or greater
    Other dependencies are listed in package.json

## Steps to run:
1) Install the dependencies.
```
$ npm install
```
2) Run the server
```
$ npm start
```
3) Go to http://127.0.0.1:3000/ to see the project in action.
